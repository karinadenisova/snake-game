# 🐍 Snake Game — 4 languages ✨

Classic Snake arcade game implemented in **Python**, **JavaScript (HTML5 canvas)**, **C++ (console)**, and **Go (terminal)**.

## 🎮 Play / Run

| Language | File | How to run |
|:---|:---|:---|
| Python | `python/snake.py` | `python snake.py` (requires `curses` / `keyboard` on Windows) |
| JavaScript | `javascript/snake.html` | Open in any browser |
| C++ | `cpp/snake.cpp` | `g++ snake.cpp -o snake && ./snake` |
| Go | `go/snake.go` | `go run snake.go` |

## 🕹️ Controls

- **Arrow keys** (or **WASD**) — move the snake
- **Q** or **ESC** — quit
- **R** — restart after game over

## 🧠 What each version shows

- **Python** – simplicity, terminal graphics with `curses`
- **JavaScript** – canvas + event loop, modern web game
- **C++** – low-level console I/O, raw input handling
- **Go** – goroutines for non-blocking input, channels
